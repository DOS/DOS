# DOS — Web3 Gaming Ecosystem

> Onboarding the next billion gamers into Web3

DOS is a high-speed, gas-free, AI-powered Layer-1 blockchain built on [Avalanche](https://www.avax.network/), purpose-built for Web3 gaming.

## Key Features

- **Zero Gas Fees** — All whitelisted smart contract functions operate without gas costs
- **Sub-Second Finality** — Transactions are confirmed instantly for real-time gameplay
- **4,500+ TPS** — High throughput with potential for further scaling
- **EVM Compatible** — Full compatibility with Ethereum tooling and smart contracts
- **AI-Powered** — Smart wallets, risk detection, automated asset management, and NPC AI agents
- **Cross-Chain** — Asset interoperability via Avalanche ICM and LayerZero

## Architecture

DOS Chain runs as a dedicated Avalanche L1 (formerly Subnet) with:

- **Consensus**: Avalanche Snowman consensus
- **VM**: Subnet-EVM (EVM-compatible)
- **Cross-chain messaging**: Avalanche ICM (Interchain Messaging)
- **Validator model**: Proof of Authority (PoA) via ValidatorManager precompile
